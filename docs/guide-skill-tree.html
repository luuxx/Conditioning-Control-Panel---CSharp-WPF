Sparkle Points

Sparkle points are the currency used to unlock skills in the Skill Tree. You earn them automatically as you level up.

  • Earn 5 sparkle points per level up
  • Points are managed server-side (source of truth)
  • Skills once unlocked cannot be refunded (unless admin force-reset)
  • At Level 50 you'll have earned approximately 245 total sparkle points
ğŸ”„ Server Sync

The server is the source of truth for your skill points. If your local count differs, the server value takes priority during sync.

Pink Rush

Pink Rush is one of the most exciting skills in the Skill Tree, offering massive XP bursts during random windows of opportunity.

  • Unlocked via the Pink Rush skill (Tier 4, 25 points)
  • ~50% chance to trigger every 5 minutes while engine is running
  • Lasts exactly 60 seconds
  • During Pink Rush: all XP gains are multiplied by 3x
  • Can trigger multiple times per session
