Dangerous Features

‚ö†Ô∏è Use With Extreme Caution

These features can lock you into sessions with limited escape options. Only use them if you fully understand the implications.

Strict Lock

What it does: Cannot close, minimize, or skip videos.

  • You will be forced to watch the entire video
  • Only the panic key can stop it (unless No Panic is also enabled)
  • A confirmation dialog requires you to acknowledge the risks

No Panic

What it does: Completely disables the panic key.

  • There will be NO way to stop the engine except:
  • Wait for scheduler to end the session
  • Force-close via Task Manager (Ctrl+Shift+Esc)
  • A double-confirmation dialog is required

Combining Both

üîí Total Lockdown

If both Strict Lock AND No Panic are enabled:

  • Videos cannot be skipped
  • Panic key doesn't work
  • Only options: wait for video to end, or Task Manager

Performance Tips

Running many features simultaneously, especially with high frequencies, multiple images, or high-resolution videos/GIFs, can be resource-intensive. If you experience performance issues, reduce the number of active features or their intensity.

Frequently Asked Questions

Why don't my GIFs animate?

Ensure GIFs are under 5MB. Very large GIFs may not animate smoothly due to memory constraints.

Can I use this on multiple monitors?

Yes! Enable "Dual Mon" in System settings. Flash images and overlays appear on all monitors. Videos play on the primary monitor. The detached avatar can be placed on any monitor.

How do I completely exit the app?

Double-tap the panic key, or right-click the tray icon ‚Üí Exit, or use the Exit button in the app.

Where do I put my files?

In the assets/ subfolders: images/, sounds/, startle_videos/. Click the üìÇ Assets button to open the folder.

Is my data sent anywhere?

The app is mostly local. Patreon features (AI Chat, Window Awareness) send data to our secure proxy server for processing, but no data is stored permanently. See the Privacy Notice in the Patreon tab.

Can I use this while gaming?

Yes, but overlays may interfere with fullscreen games. Use borderless/windowed mode for best results. The "Gamer Girl" session is designed for gaming scenarios.

How do I add trigger audio?

Place audio files in Resources/sub_audio/ with names matching your triggers (e.g., GOOD GIRL.mp3 for the "GOOD GIRL" trigger). Files are matched case-insensitively.

What does Window Awareness track?

It reads the active window name and browser tab title to provide contextual AI responses. See the Privacy Notice in the Patreon tab for full details.

How do I reset all settings?

Close the app, delete settings.json from the app folder, and restart. This resets everything to defaults (but keeps your XP/level progress).

Why does my antivirus flag the app?

Some antivirus software flags the app due to screen overlay functionality. This is a false positive. The app is verified clean on VirusTotal and is fully open source.
