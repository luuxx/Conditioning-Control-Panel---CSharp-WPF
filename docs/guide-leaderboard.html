Monthly Seasons

Seasons run on a monthly cycle, each with a themed title to keep things fresh and competitive.

How Seasons Work

  • Monthly Cycle ‚Äî Each season follows a YYYY-MM format (e.g., 2026-02)
  • Themed Titles ‚Äî Every season has a unique name (e.g., "Fucked-up February", "Jerk-it January")
  • Season Reset ‚Äî At the start of a new season, XP and level reset to 0, and stats are archived to your all-time stats

What Persists Between Seasons

  • Skill tree unlocks
  • Achievements
  • Account data

What Resets Each Season

  • XP and level
  • Session stats
  • Quest progress
  • Streak
üìù Note

Don't worry about losing progress ‚Äî your all-time stats are always preserved, and skill tree investments carry over between seasons.

Rankings

The leaderboard is sorted by total XP within the current season. Your level and title are displayed alongside your rank, and online status is shown via a heartbeat signal sent every 45 seconds.

Your display name is customizable for privacy, and you can view other users' profiles including their achievements and stats.

Rank Display Details
Position Your # rank out of all players
Level & Title Current season level with title
XP Total XP earned this season
Online Status Green dot if heartbeat within 45s
Display Name Customizable (default: platform username)

OG Badges

The "Season 0 OG" badge is awarded to all participants in the first season. This badge is retroactively applied ‚Äî even if you started before the badge system was introduced.

  • Permanent ‚Äî OG status is a permanent flag on your account
  • Visible ‚Äî Displayed as a special indicator on the leaderboard and profile
  • Automatic ‚Äî No action required to claim it
‚úÖ Automatic Recognition

If you participated in Season 0, you automatically have OG status ‚Äî no action needed!

Profile Viewer

Click any user on the leaderboard to view their profile. Profiles display level, XP, achievements, stats, and online status.

Privacy Settings

You control what others can see on your profile:

Setting Description
Display Name Customizable name shown on the leaderboard
Profile Picture Sharing Toggle whether your profile picture is visible to others
Online Status Visibility Toggle whether others can see when you're online
Discord DM Allowance Toggle whether users can send you Discord DMs

Web Leaderboard

The leaderboard is also accessible via the web at the app's website. It shows the same data as the in-app leaderboard and updates in real-time.

You can access the web leaderboard from the main docs site at leaderboard.html.
