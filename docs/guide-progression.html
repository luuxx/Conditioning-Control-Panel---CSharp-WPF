XP System

Earn XP (experience points) through various interactions in the app. XP accumulates to level you up, unlock new features, and climb the leaderboard. All base XP values below are further multiplied by your global XP multiplier from the Skill Tree.

Session XP

Sessions are the biggest XP source. The base reward depends on difficulty, and is then boosted by your level multiplier and a duration bonus for longer sessions.

Session Difficulty Base XP Notes
Easy 400 XP e.g. Morning Drift
Medium 800 XP e.g. Random Drop, Gamer Girl
Hard 1,200 XP e.g. Good Girls Don't Cum, Deep Dive
Extreme 2,000 XP e.g. Bambi Time
Remote Control 200 XP Controlled by another user
Custom (Timeline) Varies Based on duration + enabled features

Pause penalty: -100 XP per pause during a session.

Session Level Multiplier

Higher-level players earn more session XP. This multiplier kicks in at level 30 and scales up:

Level Range Multiplier
1 - 291.0x
30 - 791.0x - 1.5x
80 - 1241.5x - 2.4x
125 - 1492.4x - 3.15x
150+3.15x+

Duration Bonus

Sessions longer than 2 minutes award extra XP based on duration and level:

Duration Bonus = (minutes - 2) × (8 + level × 0.15)

Example: Level 50, 30-minute session
  = (30 - 2) × (8 + 50 × 0.15)
  = 28 × 15.5 = 434 bonus XP

💤 Idle Detection

Flash, Subliminal, and Bouncing Text XP are suppressed when you're detected as idle. Active interactions like Bubbles, Videos, and Lock Cards still award XP.

Quest XP

Quests are a major XP source that scales with your level. See the Quests guide for the full list.

Quest Type Base XP Scaling
Daily Quests 100 - 250 XP +4% per level
Weekly Quests 500 - 1,000 XP +4% per level

Quest bonuses:

  • Better Quests skill: +25% quest XP
  • Quest streak: +3% per consecutive day of daily quest completion

Streak & Milestone XP

Log in daily to build your streak. With the Milestone Rewards skill unlocked, you earn bonus XP each day:

Streak bonuses also scale with level (+3% per level).

With the Perfect Bimbo Week skill, you earn large milestone bonuses:

Milestone bonuses scale with level (+2% per level).

XP Formula (Leveling)

Each level requires 50 + (level x 20) XP to advance.

Level 1 → 2:   70 XP  (50 + 1x20)
Level 2 → 3:   90 XP  (50 + 2x20)
Level 5 → 6:   150 XP (50 + 5x20)
Level 10 → 11: 250 XP (50 + 10x20)
Level 50 → 51: 1,050 XP (50 + 50x20)
