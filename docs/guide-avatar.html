Avatar Features

Detachable Sprite

The avatar can be detached from the main window to float freely:

  • Right-click the avatar to access the context menu
  • Select Detach to make it float freely
  • Drag the detached avatar to any position on any monitor
  • Select Attach to reattach it to the main window

Speech Bubbles

The avatar displays speech bubbles in response to various events:

  • Idle chatter - When you've been away
  • Activity reactions - Context-aware with Window Awareness (Patreon)
  • Trigger phrases - In Trigger Mode
  • AI responses - When chatting (Patreon)
ðŸ’¡ Dismissing Bubbles

Click on any speech bubble to dismiss it immediately.

AI Personalities (Prompts)

Customize how your AI companion behaves and speaks using community prompts. Each companion can have a different personality assigned.

Installing Prompts

  1. Go to Settings â†’ AI Settings
  2. Click Browse Community Prompts
  3. Select and install prompts you like
  4. Click Activate to use a prompt

Assigning to Companions

Each companion can have its own AI personality:

  1. Go to the Companion tab
  2. Find the companion you want to customize
  3. Click the ðŸŽ­ (Personality) button
  4. Select a prompt to assign
ðŸ’¡ Auto-Switch

When you switch companions, the app automatically activates their assigned personality prompt. No need to manually switch!

Creating Custom Prompts

You can create your own personality prompts:

  1. Click Create New Prompt in AI Settings
  2. Give it a name and description
  3. Write your system prompt (defines how the AI behaves)
  4. Save and activate

Trigger Mode

Trigger Mode displays random conditioning phrases at set intervals.

Setting Description
Enable Trigger Mode Show random trigger phrases
Trigger Interval Seconds between triggers (10-600s)
Edit Triggers Customize trigger phrases

How It Works

  1. Avatar displays random trigger phrases from your list
  2. If Audio Whispers is enabled, matching audio clips play
  3. Audio files should be in Resources/sub_audio/ named to match triggers

For the trigger "Good Girl" to play audio, place a file named GOOD GIRL.mp3 in Resources/sub_audio/
