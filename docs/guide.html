Quick Reference

Keyboard Shortcuts

Esc Stop engine (panic key)
Esc Esc Force exit application
Ctrl+Shift+Esc Task Manager (emergency)

Supported Formats

Images .jpg, .jpeg, .png, .gif
Audio .mp3, .wav
Video .mp4, .webm, .avi

Folder Structure

assets/
├── images/
├── sounds/
├── startle_videos/
└── spirals/

XP Gains

Click image +5 XP
Pop bubble +2 XP
Attention check +10 XP
Watch video +20 XP
Daily quest 100-250 XP
Weekly quest 500-1,000 XP

Quest Schedule

Weeklies 3 active, refresh each week
