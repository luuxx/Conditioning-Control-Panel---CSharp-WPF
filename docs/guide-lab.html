Remote Control

Remote Control allows another person to remotely control your app's effects from their phone or browser. Hand over the reins and let someone else run the show.

How It Works

  1. Enable Remote Control in The Lab tab
  2. A unique session code is generated
  3. Share the code with your controller
  4. They connect via the web-based remote control page
  5. They can now trigger effects on your app in real time

Permission Tiers

You choose how much control to give. Each tier includes everything from the tier below it:

Tier What They Can Control
Light Flashes, overlays
Standard + Videos, haptics
Full + Autonomy, sessions, strict lock
