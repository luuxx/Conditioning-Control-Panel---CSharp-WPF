Installation

Download

  1. Go to the GitHub Releases page
  2. Download the latest ConditioningControlPanel-Setup.exe
  3. Run the installer and follow the prompts

Portable Version

Alternatively, download the portable ZIP version:

  1. Extract the ZIP to any folder
  2. Run ConditioningControlPanel.exe
âœ… 100% Safe

The application is open source and scanned clean on VirusTotal (0/72 detections). Some antivirus software may flag it due to the nature of screen overlays - this is a false positive.

First Launch

When you first launch the application:

  1. Folder Creation - The app creates necessary folders in your assets directory:
    • images/ - for flash images
    • sounds/ - for accompanying sounds
    • startle_videos/ - for mandatory videos
    • spirals/ - for spiral overlay GIFs
  2. Default Settings - Sensible defaults are loaded
  3. Starting Level - You begin at Level 1 with 0 XP
  4. Companion Avatar - The avatar appears attached to the main window
ğŸ“ Data Location

User data is stored in %APPDATA%/ConditioningControlPanel/ including settings, progress, and browser data.

Basic Workflow

Starting a Session

  1. Add Content - Put your images and videos in the assets folders
  2. Configure Settings - Adjust flash frequency, video settings, etc.
  3. Click START - The conditioning engine begins
  4. Interact - Click images, complete attention checks, earn XP
  5. Stop Anytime - Press the panic key (default: Escape)

The Panic Key

ğŸ”‘ Emergency Stop

Single press: Stops the conditioning engine
Double-tap (within 2 seconds): Force exits the application

Recommended First Steps

  1. Start with low frequencies (2-3 flashes per minute)
  2. Keep Clickable enabled so you can dismiss images
  3. Leave the panic key active (don't enable No Panic)
  4. Try a built-in session like "Morning Drift" to get familiar
  5. Explore the Companion Avatar features
