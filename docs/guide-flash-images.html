Overview

Flash images appear randomly on your screen based on your configured settings. They support both static images (JPG, PNG) and animated GIFs. You can make them clickable for interaction, or enable Corruption mode for an overwhelming experience.

Flash Settings

Setting Range Description
Enable On/Off Master toggle for flash images
Clickable On/Off Allow clicking to dismiss images
Corruption On/Off Clicking spawns MORE images (hydra mode)
Per Min 1-10 Flash events per minute
Images 1-15 Images shown per flash event
Max On Screen 5-20 Hard limit on simultaneous images

How It Works

  1. Timer Trigger - Every 60 / Per Min seconds, a flash event triggers
  2. Image Selection - The specified number of random images are selected from your images/ folder
  3. Random Placement - Each image appears at a random position on screen
  4. Interaction:
    • If Clickable is ON: clicking dismisses the image (+5 XP)
    • If Corruption is ON: clicking spawns 2 more images instead!
  5. Limit Enforcement - Max On Screen prevents screen flooding
üí° XP Tip

Each image click earns you +5 XP. With Corruption mode, you can farm more clicks (and XP) from a single flash event!

Corruption Mode

Corruption mode creates a "hydra effect" where clicking an image spawns two more in its place. This creates an overwhelming cascade of images that requires strategy to manage.

‚ö†Ô∏è Intense Feature

Corruption mode can quickly fill your screen with images. The Max On Screen limit prevents total screen takeover, but it can still be quite overwhelming!

Strategy Tips

  • Don't click images if you want them to stay manageable
  • Let the timer clear images naturally
  • Lower the Max On Screen value if it gets too intense
  • Use the panic key to stop everything if needed
