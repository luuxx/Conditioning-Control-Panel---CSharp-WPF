Panic Key

The panic key is your emergency stop button.

Action Result
Single press Stop the conditioning engine
Double-tap (within 2 seconds) Force exit the application

Changing the Panic Key

  1. Click the üîë Escape button (shows current key)
  2. Press any key to set it as the new panic key
  3. The button label updates to show the new key
‚ö†Ô∏è No Panic Mode

Enabling "No Panic" completely disables the panic key. Your only options to stop will be waiting for the scheduler to end, or using Task Manager (Ctrl+Shift+Esc).

Audio Settings

Setting Range Description
Master 0-100% Overall volume level for all app audio
Audio Duck On/Off Lower other apps' audio during video playback
Duck % 0-100% How much to reduce other audio

How Ducking Works

  1. A video starts playing
  2. System volume for other apps reduces by Duck %
  3. Video plays at the Master volume level
  4. When video ends, other audio restores to normal

Browser Data

Browser cookies, cache, and history are stored in the browser_data/ folder. Delete this folder to clear all browser data.

üîç Zoom Level

The browser defaults to 75% zoom for a compact view. Use Ctrl+Scroll to adjust zoom level.

Multi-Monitor Setup

Enable Dual Mon in System settings to use multiple monitors:

  • Videos play on primary monitor only
  • Flash images appear on all monitors
  • Overlays (spiral, pink filter) cover all monitors
  • Detached avatar can be placed on any monitor
üí° Gaming Tip

When gaming, keep your game on one monitor and the avatar/overlays on another for a less intrusive experience.

Presets

Save and quickly load different setting configurations.

Using Presets

  1. Configure your settings as desired
  2. Click Save Preset and give it a name
  3. Load presets anytime from the preset dropdown
  4. Delete presets you no longer need

Resetting Everything

  1. Close the application
  2. Delete settings.json (resets to defaults)
  3. Delete browser_data/ folder (clears browser)
  4. Restart the application
