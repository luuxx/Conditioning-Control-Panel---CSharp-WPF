Overview

Subliminal messages are brief text flashes that appear on screen, optionally accompanied by whispered audio. They reinforce conditioning phrases at a pace fast enough to register subconsciously.

Text Settings

Setting Range Description
Enable On/Off Master toggle for subliminal messages
Per Min 1-30 Messages displayed per minute
Frames 1-10 Display duration (lower = faster flash)
Opacity 10-100% Text visibility/transparency

Display Behavior

  • Messages appear centered on screen
  • Text is styled with a glowing pink/purple effect
  • Lower Frames value = shorter, more "subliminal" flash
  • Higher Frames value = longer, more readable display
üí° Intensity Tips

Subtle: Opacity 30-50%, Frames 2-3
Noticeable: Opacity 70-90%, Frames 5-7
Obvious: Opacity 100%, Frames 8-10

Audio Whispers

Audio whispers play accompanying voice clips when subliminal messages appear.

Setting Description
Enable Whispers Toggle whispered audio playback
Volume Whisper volume (0-100%)

How It Works

  1. A subliminal message triggers (e.g., "Good Girl")
  2. The app looks for a matching audio file (e.g., GOOD GIRL.mp3)
  3. If found, the audio plays at the configured volume
  4. If not found, only the text displays

Haptic Synchronization

Subliminal messages can trigger haptic feedback on connected devices.

üì≥ Supported Devices

Haptic integration works with Lovense devices and any device compatible with Buttplug.io. Configure haptic settings in the Settings tab.

How It Works

  • When a subliminal message displays, a brief vibration pulse triggers
  • Pulse intensity and duration are configurable
  • Creates a physical association with the conditioning phrases
